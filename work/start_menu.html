VS Code 실행 메뉴

VS Code의 현재 폴더에서 작업합니다. 하단폴더는 project폴더가 있습니다.

📁 네이버 지식인 크롤링

project/data_crawl/crawl_app.py

네이버 지식인 데이터를 수집하여 분석할 수 있는 프로그램입니다.

python project/data_crawl/crawl_app.py

VS Code에서 실행하려면:

  1. VS Code에서 파일 열기
  2. F5 키 또는 실행 버튼 클릭
