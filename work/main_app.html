주요 기능

1

카드 데이터 자동 분류

가맹점명 기반 카테고리 분류 및 부가세 계산을 자동으로 수행합니다.

2

부가세 계산 자동화

카테고리별 부가세 공제 여부를 확인하고 자동으로 계산합니다.

3

데이터 시각화

월별, 카테고리별 지출 패턴을 그래프로 시각화합니다.

4

세금 신고 안내

홈택스 신고 방법 및 세금 납부 절차를 안내합니다.

5

절세 팁 제공

개인사업자를 위한 맞춤형 절세 팁을 제공합니다.

6

청년 창업자 지원

청년 창업자 특화 세제 혜택 및 지원 정책 정보를 제공합니다.

사용 방법

  1. 데이터 준비: 롯데카드 엑셀 파일을 준비합니다.
  2. 데이터 전처리: '데이터 전처리' 도구를 실행하여 카드 데이터를 분석합니다.
  3. 데이터 다운로드: 전처리된 데이터를 CSV 파일로 다운로드합니다.
  4. 부가세 신고: '부가세 신고 도우미'를 실행하여 전처리된 데이터를 업로드합니다.
  5. 신고서 작성: 분석 결과를 바탕으로 홈택스에서 신고서를 작성합니다.
